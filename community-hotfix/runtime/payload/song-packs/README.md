# Optional song packs

The installer supports a `FixPlusSongs` edition, but a preview pack must include:

- `preview/pack.json` with `id`, `version`, and a `files` array.
- `preview/files/` containing the addon-relative overlay files.
- A clear provenance and attribution record for every chart, cover, and audio file.

The current preview is built from files attached to a chart-submission Issue. Keep the submitter and Issue link in `ATTRIBUTION.md`.
